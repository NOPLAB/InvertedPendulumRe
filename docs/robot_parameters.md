# ロボット機体データ

台車型倒立振子ロボットの物理パラメータ。

## モーターパラメータ（データシート / 実測値）

| 記号 | パラメータ           | 値       | 単位    |
| ---- | -------------------- | -------- | ------- |
| Kt   | トルク定数           | 0.01857  | N*m/A   |
| Ke   | 逆起電力定数         | 0.01857  | V*s/rad |
| Ra   | 電機子抵抗           | 32.4     | Ohm     |
| La   | 電機子インダクタンス | 2.955e-3 | H       |
| G    | ギア比               | 6.666    | -       |

モーターは左右2個搭載（並列駆動）。

## 機械系パラメータ（実測値）

| 記号 | パラメータ                 | 値     | 単位   |
| ---- | -------------------------- | ------ | ------ |
| M    | 台車質量                   | 0.57   | kg     |
| m    | 振子質量                   | 0.0303 | kg     |
| l    | 振子長さ（回転軸から重心） | 0.15   | m      |
| r    | タイヤ半径                 | 0.0255 | m      |
| g    | 重力加速度                 | 9.81   | m/s^2  |
| Iw   | タイヤ慣性モーメント       | ~0     | kg*m^2 |

## 機械系パラメータ（同定値）

パラメータ推定により取得。

| 記号   | パラメータ                       | 値         | 単位      |
| ------ | -------------------------------- | ---------- | --------- |
| Jp     | 振子慣性モーメント（回転軸周り） | 2.2725e-4  | kg*m^2    |
| bx     | 台車粘性摩擦係数                 | 0.0        | N*s/m     |
| btheta | 振子粘性摩擦係数                 | 1.67926e-3 | N*m*s/rad |
| Dz     | 不感帯                           | 2.40146    | -         |
